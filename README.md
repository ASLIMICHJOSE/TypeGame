# TypeBlaze Typing Arena

An intense, high-energy typing speed speed-run game designed to master typing agility with responsive visual theme accents, precise feedback, statistics, and rank progression.

## Features

- **5 Progressive Difficulty Arenas**: 
  - 🌱 **Ember** (≤ 30 WPM)
  - 🔥 **Blaze** (30 – 50 WPM)
  - ⚡ **Storm** (50 – 70 WPM)
  - 💀 **Inferno** (70 – 90 WPM)
  - 👑 **Legend** (90+ WPM)
- **Fluid Mechanics**: Real-time WPM, Accuracy, and Combo calculations with combo milestones, shake animations on errors, and custom countdown transitions.
- **Client-Side Routing**: Handled natively using `react-router-dom` for persistent URL structures across Home, Play, Results, Leaderboard, and Badges screens.
- **Persistent Progress**: Achievements, total earned XP, tier progress, and best statistics saved locally.
- **Leaderboard Integration**: Play and compete against real-time simulated AI profiles (e.g. PyroByte, AshKnight).
- **Badge Achievement System**: 12 custom badges (e.g., Unbreakable, Sharpshooter, Stormcaller) to track milestones.

## Technologies Used

- **React** (v19)
- **Vite**
- **React Router** (v6)
- **Vanilla CSS** with fire and neon glow variables, CSS keyframes animations, and modular responsive grids.

## Getting Started

### Prerequisites

- Node.js (v18+)
- npm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/ASLIMICHJOSE/TypeGame.git
   cd TypeGame
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Launch development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
